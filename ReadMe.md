# reliance-test-host
A demo of using reliance from a host C app

## Building
Prerequisites:
- [reliance package manager](https://github.com/RepComm/reliance)
- cc compiler (gcc if you use build.sh)

1. Install dependencies
```bash
reliance install
```

2. Build
```bash
./build.sh
```

3. Run
```bash
./run.sh
```
